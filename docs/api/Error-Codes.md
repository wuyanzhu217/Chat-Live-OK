# 统一错误码

> 版本：V1.0  
> 响应格式见 [API-Overview.md](API-Overview.md)

---

## 1. 成功

| code | message |
|------|---------|
| 0 | ok |

---

## 2. 通用（1000–1999）

| code | HTTP | 说明 |
|------|------|------|
| 1001 | 400 | 参数无效 |
| 1002 | 401 | 未登录或 token 无效 |
| 1003 | 401 | access_token 过期 |
| 1004 | 403 | 无权限 |
| 1005 | 404 | 资源不存在 |
| 1006 | 409 | 资源冲突 |
| 1007 | 429 | 请求过于频繁 |
| 1008 | 500 | 内部错误 |
| 1009 | 501 | 功能未实现（V2 预留接口） |

---

## 3. 用户 / 好友（2000–2999）

| code | 说明 |
|------|------|
| 2001 | 用户名已存在（业务库同步） |
| 2004 | 用户不存在 |
| 2101 | 已是好友 |
| 2102 | 好友请求已存在 |
| 2103 | 不能添加自己为好友 |
| 2104 | 好友请求不存在 |

---

## 4. 会话 / 消息（3000–3999）

| code | 说明 |
|------|------|
| 3001 | 会话不存在 |
| 3002 | 非会话成员 |
| 3003 | 消息不存在 |
| 3004 | 不支持的消息类型 |
| 3005 | 图片文件过大 |
| 3006 | 上传失败 |

---

## 5. 通话（4000–4999）

| code | 说明 |
|------|------|
| 4001 | 通话不存在 |
| 4002 | 对方忙线 |
| 4003 | 非法通话状态 |
| 4004 | 非通话参与方 |
| 4005 | 不能呼叫自己 |
| 4006 | 非好友，不可呼叫 |

---

## 6. 直播（5000–5999）

| code | 说明 |
|------|------|
| 5001 | 直播间不存在 |
| 5002 | 非主播，无权操作 |
| 5003 | 直播间状态不允许该操作 |
| 5004 | 推流鉴权失败 |
| 5005 | 已有进行中的直播 |

---

## 7. WebSocket error 事件

```json
{
  "event": "error",
  "data": {
    "code": 1002,
    "message": "token expired"
  }
}
```

---

## 8. 扩展规则

- 新增错误码仅追加，不修改已有 code 含义
- 各模块预留 100 个号段空位供细分
